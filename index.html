Juros Simples

Ideal para empréstimos de curto prazo onde os juros não rendem sobre juros acumulados.

FV = PV × (1 + i × n)
FV: Montante Final
PV: Capital Inicial
i: Taxa (decimal)
n: Período

Juros Compostos

O modelo padrão do mercado financeiro moderno. O capital cresce exponencialmente.

FV = PV × (1 + i)n
FV: Montante Final
PV: Capital Inicial
i: Taxa Juro
n: Tempo

Isolando o Tempo (n)

Use esta fórmula para descobrir quanto tempo deve esperar para atingir um objetivo financeiro.

n = log(FV / PV) / log(1 + i)

Isolando a Taxa (i)

Descubra qual a rentabilidade necessária para o seu investimento.

i = (FV / PV)1/n - 1
